# chromebox
 Chrome Custom Tab Support for Android 

## Getting Started

```yaml
  # Android CustomTabs
  chromebox:
    git:
      url: https://github.com/nghuyy/chromebox.git
      ref: main
```

 using:
 ```dart
    import 'package:chromebox/chromebox.dart';

    Chromebox.startTab("https://google.com.vn");
 ```
